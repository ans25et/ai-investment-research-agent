# AI Investment Research Agent

AI-powered investment research platform that helps users analyze public companies, summarize financial information, compare investment opportunities, and generate structured investment insights through an intuitive web interface.

## 🌐 Live Demo

https://ai-investment-research-agent.vercel.app

## 💻 GitHub Repository

https://github.com/ans25et/ai-investment-research-agent

---

## Overview

The AI Investment Research Agent streamlines the investment research process by combining modern web technologies with AI-powered analysis. The application is designed to help users quickly gather company information, review financial data, and generate organized research summaries that support investment decision-making.

---

## Features

- 🤖 AI-powered company analysis
- 📈 Investment research and financial summaries
- 📊 Company comparison tools
- 📑 Structured research reports
- ⚡ Responsive interface for desktop, tablet, and mobile
- 🔒 Secure backend architecture
- 🌐 Modern React frontend with Express backend

## Getting Started

### Clone the repository

```bash
git clone https://github.com/ans25et/ai-investment-research-agent.git
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```
